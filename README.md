# Mouse Highlight

A [Nexus](https://raidcore.gg/Nexus) port of the [TacO](https://www.gw2taco.com/2015/01/feature-list.html) addon that
renders a full-screen crosshair
aligned to the mouse cursor.

![img.png](docs/preview.png)

## Options

- Either show always, in combat only or only out of combat
- Customize color and thickness of the crosshair
